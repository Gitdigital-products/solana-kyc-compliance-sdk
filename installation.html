Installation

Get your Solana KYC SDK up and running quickly.

Using NPM

npm install @gitdigital/solana-kyc-sdk

Using Yarn

yarn add @gitdigital/solana-kyc-sdk

Importing in Your Project

import { KYCClient } from "@gitdigital/solana-kyc-sdk";
