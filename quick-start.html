Quick Start

Follow these steps to integrate KYC verification into your app.

Step 1: Import and Configure


import { KYCClient } from "@gitdigital/solana-kyc-sdk";

const kyc = new KYCClient({
  apiKey: "YOUR_API_KEY",
  network: "devnet",
  callbackUrl: "https://yourapp.com/kyc-callback"
});

Step 2: Start Verification


const userId = "123456";
await kyc.startVerification(userId);

Step 3: Listen for Status Updates


kyc.on("verified", (userId) => {
  console.log(`${userId} is verified!`);
});
