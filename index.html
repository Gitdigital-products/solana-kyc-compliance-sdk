Solana KYC Compliance SDK

An open-source SDK for enforcing KYC/AML compliance directly at the token level on Solana using Token Extensions. Includes a Rust on-chain program, TypeScript SDK, and Compliance Registry for institutional-grade Real-World Asset (RWA) issuance.

What It Is

KYC & Identity Workflows

Provides modular tooling for identity verification workflows and compliance-aware Solana program integration.

SDK, Not Custodian

Audit-Ready by Default

Built with logging, controls, and comprehensive documentation as first-class citizens for enterprise use.

Design Philosophy

  • Compliance-aware, not compliance-blocking – Integrates KYC/AML support without killing user experience.
  • Blockchain-native – Designed around Solana’s execution model, RPC dependencies, and ecosystem realities.
  • For Builders – Built for Web3 startups, regulated DeFi platforms, wallets, marketplaces, and enterprises deploying on Solana.
