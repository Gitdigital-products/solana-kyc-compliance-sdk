## Reporting a Vulnerability

We take the security of the Solana KYC Compliance SDK seriously. We appreciate the efforts of security researchers and the open-source community to report vulnerabilities responsibly.

### How to Report

If you discover a security vulnerability within this repository, please **DO NOT** open a public GitHub issue.

Instead, please send a detailed email to **security@gitdigital.com** (replace with the actual security email address if different).

In your email, please include:
* A clear and concise description of the vulnerability.
* Steps to reproduce the vulnerability.
* The affected version(s) of the SDK.
* Any proof-of-concept (PoC) code or scripts.

### Our Commitment

We will acknowledge receipt of your report within **48 hours** and provide a detailed response to your reported vulnerability within a reasonable timeframe, typically within **7 days**. We will keep you informed of our progress in fixing the issue and will credit you for your discovery if you wish, once the vulnerability is addressed and a new release is made.
