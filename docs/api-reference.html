API Reference

This document describes the primary interfaces exposed by the Solana KYC Compliance SDK.

Initialize SDK

initializeKycSdk({ provider: "exampleProvider", environment: "production" });

Start Verification

startVerification({ userId, jurisdiction, verificationLevel });

Check Verification Status

getVerificationStatus(userId);

Consume On-Chain

Applications may pass abstract verification signals to Solana programs. No identity data is included.

assertVerified(walletAddress, scope);
